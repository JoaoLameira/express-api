# Express API

Express api with Typescript, Prisma and Postgres.

## Dependencies

- Typescript
- Prisma CLI
- Express

```
npm i express typescript ts-node @types/node prisma --save-dev
```

## Prisma SDK

```
npm i @prisma/client --save
```

### Migrations

run migration and give it a name

```
npx prisma migrate dev --name init
```
